============================================
BioGraPy - Biological Graphic tool in Python
============================================

.. toctree::
   :maxdepth: 1
   
   
BioGraPy is a tool to draw biological diagrams. It makes use of the powerful
matplotlib_ plotting library to draw any kind of information along a biological
sequences. BioGraPy is tightly integrated with BioPython_, but BioPython is not 
required if you want to draw very simple diagrams. 
Take a look at the :ref:`tutorial` to have an idea of what you can do with BioGraPy.

.. _BioPython: http://www.biopython.org

.. _matplotlib: http://matplotlib.github.com

Download
--------
You can download BioGraPy in either zip_ or tar_ formats.

.. _zip: http://github.com/apierleoni/BioGraPy/zipball/v1.0b

.. _tar: http://github.com/apierleoni/BioGraPy/tarball/v1.0b

You can also clone BioGraPy with Git by running:

::

    $ git clone git://github.com/apierleoni/BioGraPy

Get the source code (or make your own fork) on GitHub : `apierleoni/BioGraPy 
<http://github.com/apierleoni/BioGraPy>`_
